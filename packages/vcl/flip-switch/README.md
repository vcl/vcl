# [VCL](https://vcl.github.io/) flip-switch

A switch that can be toggled between two states.

## Features

Two states between which can be toggled.

## Usage

[basic example](/demo/example.html)

- `flip-switch`

- `flip-switch-track`
- `flip-switch-knob`

## Modifiers

- `disabled`
- `flip-switch-pressed`
